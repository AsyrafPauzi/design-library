# Input

## Default
- Background: white
- Border: 1px `color.border.default` (#E2E8F0)
- Border radius: 8px
- Padding: 12px 16px
- Height: 44px
- Font: body (16px)
- Text color: `color.text.primary`

## Label
- Above input, 8px gap
- Style: label (14px medium)
- Color: `color.text.primary`

## Placeholder
- Color: `color.text.secondary`

## Focus
- Border: `color.border.focus` (#2563EB)
- Optional: 2px focus ring in primary light

## Error
- Border: `color.status.error` (#DC2626)
- Helper text below in error color, 4px gap

## Usage
- Stack label + input + helper with 8px gaps
- 16px gap between different fields

## Do not
- Put label inside the field (use placeholder only for hints)
- Use height less than 44px
