# Button

## Variants

### Primary
- Background: `color.primary.default` (#2563EB)
- Text: `color.text.inverse` (white)
- Padding: 12px 24px
- Border radius: 8px (`borderRadius.md`)
- Min height: 44px
- Font: label style (14px medium)

### Primary Hover
- Background: `color.primary.hover` (#1D4ED8)

### Secondary
- Background: white
- Border: 1px `color.primary.default`
- Text: `color.primary.default`

### Disabled
- Background: `color.background.muted`
- Text: `color.text.disabled`
- No pointer interaction

## Usage
- One primary button per screen section
- Use secondary for cancel / back actions
- Full width on mobile forms

## Do not
- Use custom colors
- Use border radius other than 8px
- Make buttons shorter than 44px
