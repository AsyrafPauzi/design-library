# Design Library

Design system template for **Figma + Tokens Studio + Figma AI**.

**Repo:** https://github.com/AsyrafPauzi/design-library

## Setup

Follow the checklist → **[GUIDE.md](./GUIDE.md)**

## Daily use

1. Open `AI-INSTRUCTIONS.md`
2. Copy the prompt → paste in Figma AI
3. Describe your screen

```
[paste AI-INSTRUCTIONS]

Create a login screen with email, password, and login button.
```

## What's inside

```
tokens/          colors, fonts, spacing (edit or sync via Tokens Studio)
components/      button, input, card specs
AI-INSTRUCTIONS.md   copy into Figma AI every time
GUIDE.md         step-by-step checklist
```
